# ⌚ SmartFit E-commerce Landing Page

Une interface moderne et responsive pour une boutique de montres connectées premium.

## 🚀 Fonctionnalités
- Design épuré et "Mobile-First".
- Gestion dynamique du panier (JavaScript).
- Formulaire de commande interactif.
- Structure de fichiers organisée selon les standards professionnels.

## 🛠️ Technologies utilisées
- **HTML5 / CSS3** (Flexbox & Grid)
- **JavaScript** (ES6+)
- **Git/GitHub** pour le versioning

## 📁 Structure du projet
- `/assets/img` : Ressources visuelles et icônes.
- `/assets/css` : Feuilles de style.
- `/assets/js` : Logique applicative.
